# Preview Framework

## Objective
Turn an upcoming event into a clear pre-event decision.

## Four mandatory questions
1. Is the chart still clean enough to carry binary event risk?
2. Is the event still underpriced, or has the crowd already moved ahead of it?
3. Has the flow over the last 3-10 sessions been pre-positioning?
4. If the event fails, where is the stop or invalidation?

## Action Mapping
- PRE-POSITION SMALL: clean base + confirming flow + open upside
- WAIT FOR POST-EVENT TRIGGER: the thesis is fine but price has already run ahead or is close to resistance
- WATCH ONLY: the event matters, but the edge is not clear yet
- NO TRADE: binary risk is larger than the remaining upside

## Notes
- Separate a good event from a good entry
- Do not use vague expectations to justify a bad chart
