# Checklist for Reviewing Leader Stock Lists

## 1. Context Review
- Summarize current market context: macro, sector, and liquidity factors.
- Note any market events or news relevant to the stocks/groups under review.

## 2. Individual Stock Analysis
For each stock:
- Assess technical signals (price action, volume, support/resistance).
- Evaluate recent fundamental/news catalysts.
- Quantify the stock's impact on index/sector (market cap, liquidity, sector weight).

## 3. Comparison with Prior Leader List
- Cross-check whether each analyzed stock appeared on the previous leader (leading stock) list.
- For omissions, objectively assess if the stock met leader criteria at that time (e.g., volume surge, price strength, sector rotation).

## 4. Error Identification and Lesson Extraction
- Clearly state any errors or omissions in the prior leader list.
- Analyze the root cause: data lag, subjective bias, outdated screening rules, etc.
- Document specific lessons or rule adjustments to prevent recurrence.

## 5. Actionable Recommendations
- For each stock: state whether it should now be on the leader list (with rationale).
- Suggest updated leader selection criteria if patterns of omission are detected.
- Provide clear trading/portfolio implications where needed (optional).

## 6. Documentation
- Save the review as a template for future sessions.
- Note any changes to the leader screening process for ongoing improvement.

Use this checklist after major market moves, leader list updates, or when significant stocks exhibit new leadership behavior.
