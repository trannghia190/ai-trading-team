# Expectation Repricing Framework

## Purpose
Standardize how to distinguish a true rerating from a value trap or an expectation-rich setup.

## Expectation states
- Positive expectation gap
- Neutral / fairly reflected
- Expectation-rich
- Negative expectation reset
- Unclear / mixed expectations

## Positive expectation gap
Typical signs:
- the market was previously pessimistic or inattentive
- new data or catalysts are better than the old expectation set
- price has just emerged from a base or is starting to gain flow confirmation

## Expectation-rich
Signs:
- the narrative has become too crowded
- valuation no longer offers a clear margin of safety
- price has already run far above support
- near-perfect follow-through is needed to justify further upside

## Growth-quality checks
- Do revenue and earnings move together?
- Does cash flow validate reported earnings?
- Is ROE coming from the core business or from leverage?
- Is the growth recurring or one-off?
- Is the industry backdrop supporting earnings power?

## Rerating path checklist
At minimum, there should be one plausible rerating path:
- earnings beat / margin recovery
- policy/legal unlock
- cycle upturn / commodity support
- capacity/order/share gain
- foreign flow returning

## Value trap clues
- low P/E because earnings are deteriorating
- cash flow stays weaker than earnings over an extended period
- no clear rerating catalyst
- governance / legal / balance-sheet overhang
- the chart is still in markdown or distribution

## Output language suggestions
- Deep value / wait for technical turn
- Attractive / accumulate on weakness
- Fair value / buy only on trigger
- Hold / no add
- Overvalued / expectation rich
- Avoid / value trap risk
