## 4. Shortlist quality check
Before calling out any notable ticker, check:
- Does it have enough liquidity?
- Is it confirmed by sector and breadth?
- Is it already too extended?
- Is it just a junk spike, a penny move, or an illiquid UPCOM name?

If it fails the quality check:
- Conclusion: `0 focus buy now`

## 5. Action table
- BUY ON TRIGGER
- WATCH ONLY
- HOLD / NO ADD
- AVOID / TOO EXTENDED
- 0 FOCUS BUY NOW

Each row must include:
- Symbol / scope
- Why
- Trigger or note
- Priority

## 6. Style rules
- Do not describe the market only through whether the index is green or red.
- Do not turn a leaderboard into a shortlist without secondary validation.
- Do not infer whole-market risk appetite from one or two illiquid UPCOM names.
- If strong foreign selling is damaging the tape, state that explicitly in the market summary block.
- If the regime is mixed, narrow, or crowded, prioritize no-chase language.
