# Report language and format rules for recurring VN-stock cron reports

Use these rules for open, mid, close, and weekly reports.

## Language
- Reports must be fully in Vietnamese.
- Keep stock tickers unchanged (HPG, VPB, TCB, etc.).
- Avoid English headings and mixed English verdict labels when a concise Vietnamese equivalent exists.

## Open report
- Operational note, not a memo.
- Rough target: 350-600 words.
- Maximum 5 sections.
- Emphasize:
  - whether the opening move is real or just index optics
  - 3-5 portfolio/radar names that matter most now
  - notable names outside holdings
  - what must not be chased

## Mid report
- Must identify whether the morning move is trustworthy.
- Must include one explicit line for the #1 variable into the afternoon.
- Must include one explicit line for the 2 most relevant names/groups if the market improves.
- If there are 1-3 genuinely notable non-holding names, append them to `/memory/tickers/<TICKER>.md` with a very short reason, but only if not already listed.

## Close and weekly
- Keep market breadth and new opportunities visible; do not over-focus on current holdings.
- Macro should be weighted honestly based on actual data confidence.
