# VN Fundamental Valuation

## When to use
- Determine whether the stock is cheap or expensive versus its own history and sector
- Calculate a price target for Bull/Bear debate
- Evaluate margin of safety before recommending a buy

## Data to collect
1. Trailing-12-month EPS plus projected FY+1 EPS
2. Book value per share (BVPS)
3. EBITDA + Net Debt
4. Current price plus shares outstanding

## Formulas

### P/E Analysis
- Current P/E = Price / TTM EPS
- VN benchmark: VN-Index P/E is usually around 12-16x, depending on the cycle
- Price target = Target P/E x Forecast EPS
- Upside/Downside = (Target - Current price) / Current price

### P/B Analysis
- P/B = Price / BVPS
- VN banks: fair range 1.2-2.5x
- Steel/manufacturing: fair range 0.8-1.5x
- Real estate: fair range 1.0-2.0x

### EV/EBITDA
- EV = Market cap + Net debt - Cash
- <6x: cheap | 6-10x: fair | >10x: expensive in the VN context

### Simple DCF (3 years)
- WACC = 13–15% (risk premium VN)
- Terminal growth g = 3–5%
- Terminal value = FCF₃ × (1+g) / (WACC − g)
- Intrinsic value = PV(FCF₁ + FCF₂ + FCF₃) + PV(TV)

## Sample output
• TTM P/E: 12x | 5Y avg: 10x | Sector: 11x -> modest premium of about 9%
• P/B: 1.8x | Peer avg: 1.5x -> more expensive than peers
• P/E-based target: 11x x forecast EPS of 3,500 VND = 38,500 VND (+12% upside)
• Valuation conclusion: [Cheap / Fair / Expensive]
